Overview

NetworkComponentEntity is the superclass of NodeEntity and AcrEntity belonging to a NetworkEntity.

Instance Method Details

cyclicWays

Purpose: returns a collection of cyclic ways . Each cyclic way is an ordered collection of arcs or nodes, starting with the receiver element. An empty collection means that the element doesn't belong to any cyclic way. But one element may belong to several cyclic ways

downstreamWay

Purpose: returns the shortest way from this element towards the down stream direction (taking into account the orientation of the network). The way is an ordered collection of elements, starting with the receiver node. Be careful, if there are 2 (or more) shortest ways, returns only the first way of the sortedCollection. You may use downstreamWays that returns all the ways then compare them.
