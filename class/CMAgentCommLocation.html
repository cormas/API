Overview

AgentCommLocation is a class which subclass instances need to have the functionalities of AgentComm and AgentLocation

Instance Variables: acquaintancesis a set of other communicating entities to communicate with. agentObsis an instance of the interface agent (on the observer) channelis used to receive, store and deliver messages processed by communication agents mailBoxan orderedCollection to store the received messages.

Methods

messageFromChannel: will receive as argument an instance of a subclass of Msg. It will be stored in the mailBox at last position. nextMessage returns the first message in the mailBox and remove it from the mailBox. sendMessage Synchronously: will ask the channel to store the message. It will be delivered at the end of the current time step. sendmessageSynchrounously: will ask the channel to deliver the message immediately.
